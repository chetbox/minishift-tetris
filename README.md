# Minishift Tetris

A Tetris implements for a chain of 8x8 Minishift LED displays

## Installation

        $ pip install minishift numpy pygame
        $ python tetris.py NUMBER_OF_MINISHIFTS

## How to play

Key                   | Action
--------------------- | ----------
:arrow_left:          | Move left
:arrow_right:         | Move right
:arrow_up: or `Space` | Rotate
:arrow_down:          | Drop
